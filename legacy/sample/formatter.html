How It Works

Predefined Formatters

  • currency - Formats as $1,234.56 with thousand separators
  • percent - Formats as 12.34% (multiplies by 100)
  • number - Formats with 2 decimal places
  • integer - Rounds to whole number
  • text - Converts to string

Focus/Blur Behavior

For input and textarea elements with data-format attribute:

  • On Focus: Shows raw value for easy editing (e.g., 1234.56)
  • On Blur: Shows formatted value (e.g., $1,234.56)
  • While Typing: Raw value is maintained, preventing cursor jumping

Usage

<!-- Input with currency formatter -->
<input data-cell="A1" data-format="currency" value="1234.56">

<!-- Display with percent formatter -->
<span data-cell="B1" data-formula="A1*0.1" data-format="percent"></span>

<!-- Read-only input -->
<input data-formula="A1+B1" data-format="currency" readonly>

Custom Formatters

You can also register custom formatters via JavaScript:

$('#myform').calx({
    formatters: {
        phone: {
            format: function(value) {
                // Format as (123) 456-7890
                var cleaned = ('' + value).replace(/\D/g, '');
                var match = cleaned.match(/^(\d{3})(\d{3})(\d{4})$/);
                if (match) {
                    return '(' + match[1] + ') ' + match[2] + '-' + match[3];
                }
                return value;
            }
        }
    }
});
